# GNOME (GNU Network Object Model Environment)
**GNOME** is a **desktop environment**.

A desktop environment provides:  
- Windows.
- Panels / docks.
- Settings app.
- File manager.
- System menus.
- Overall look and feel.

# X11 and Xorg
X11 (X Window System) is a **display server protocol**.

It defines how:  
- Applications draw windows.
- Keyboard and mouse input are handled.
- Windows are managed on screen.

Xorg (X.Org Server) is the most common implementation of X11.

# Wayland
Wayland the modern replacement of X11.

Historically, GNOME ran on X11. Modern GNOME usually runs on Wayland by default, but many systems still allow “GNOME on X11” as a login option.
